### 초기 세팅

    1. 설치
       1. npm install
       2. npm install react-redux @reduxjs/toolkit
       3. npm install --save react-router-dom
    2. 절대 경로 설정
       1. 참고문헌: https://create-react-app.dev/docs/importing-a-component/#absolute-imports
